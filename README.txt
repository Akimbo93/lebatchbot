LE BATCH — MVP

Как запустить бота:

1. Установи Python 3.10 или выше
2. Установи зависимости:
   pip install -r requirements.txt

3. В main.py замени строку YOUR_TOKEN_HERE на токен от BotFather

4. Запусти бота:
   python main.py

Если хочешь запустить на сервере:
- Зарегистрируйся на render.com
- Создай новый Web Service
- Загрузи файлы проекта
- В команду запуска впиши:
   python main.py

LeBatch готов к нюху.
